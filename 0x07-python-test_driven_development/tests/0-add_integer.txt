# 0-add_integer.py

======================
THow to use 0-add_integer.py module
======================

Using add_integer
-------------------

This module creates a function to add to integers
First import the function from the module

    >>> add_integer = __import__('0-add_integer').add_integer

Now use it:

    >>> add_integer(1, 2)
    3

    >>> add_integer(7, 20)
    27

When the args are floats

    >>> add_integer(1.5, 2.8)
    3

    >>> add_integer(6.3, 8.5)
    14


If they are combined floats and integers

    >>> add_integer(3.2, 2)
    5

    >>> add_integer(4, 1.7)
    5

Edge cases
When one arg is not int or float

    >>> add_integer("Hello", 2)
    Traceback (most recent call last):
	TypeError: a must be an integer

    >>> add_integer(1, "Holbies")
    Traceback (most recent call last):
	TypeError: b must be an integer
